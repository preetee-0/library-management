# 📚 Library Management System

A comprehensive Library Management System built with Laravel to streamline book borrowing, returns, and student management.

## ✨ Features

- 📖 **Book Management**: Add, edit, delete, and search books with image upload support
- 👤 **Student Management**: Register and manage student records
- 📋 **Borrow & Return**: Easy book borrowing and return with due dates
- 💰 **Fine Management**: Automatic fine calculation for late returns
- 🔐 **Authentication**: Secure login and registration system
- 📊 **Dashboard**: Overview of total books, borrowed books, and pending returns
- 🎯 **Waitlist**: Students can join waitlists for popular books
- 📱 **Responsive**: Fully responsive design using Bootstrap

## 🛠️ Technologies Used

- **Laravel** - PHP Framework
- **MySQL** - Database
- **Bootstrap 5** - Frontend Framework
- **JavaScript** - Interactive elements
- **Composer** - PHP dependency management
- **Vite** - Asset bundling
- **SweetAlert2** - Beautiful alerts and modals

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

- PHP >= 8.0
- Composer
- MySQL
- Node.js & npm
- Git
